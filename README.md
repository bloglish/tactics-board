# ⚽ 축구 전술판 · Football Tactics Board

무료 온라인 축구 전술판. 설치 없이 브라우저에서 바로 사용 가능합니다.

**[→ 바로 사용하기](https://bloglish.github.io/tactics-board/)**

---

## 기능

- **포메이션 설정** — 4-3-3, 4-4-2, 4-2-3-1, 3-5-2, 5-3-2 등 자주 쓰는 포메이션 제공
- **선수 배치** — 드래그로 자유롭게 이동, 클릭으로 등번호 수정
- **홈/어웨이 색상** — 유니폼 색상 직접 설정
- **드로잉 도구** — 화살표 · 존 · 펜(자유 드로잉)
- **피치 방향** — 가로형(기본, 데스크톱) / 세로형(기본, 모바일) 전환
- **줌** — 휠 스크롤 또는 핀치 줌
- **축구 · 풋살 · 7인제** 모두 지원
- **모바일 완벽 지원** — 터치 드래그, 핀치 줌

## 조작법

| 동작 | 기능 |
|------|------|
| 선수 드래그 | 선수 이동 |
| 선수 클릭 | 등번호 수정 |
| 빈 공간 드래그 | 화살표 또는 존 또는 펜 (툴 선택에 따라) |
| 우클릭 드래그 | 존 생성 |
| 스크롤 / 핀치 | 줌 인·아웃 |
| Esc | 현재 동작 취소 |

## 기술 스택

순수 HTML · CSS · JavaScript (단일 파일, 외부 의존성 없음)
